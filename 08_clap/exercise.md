# 🎯 Упражнение: CLI-калькулятор

## Задание

Создай CLI-программу `calc` с помощью clap, которая:

### Базовые требования
1. Принимает **два числа** как позиционные аргументы (a и b)
2. Принимает флаг **`--operation` / `-o`** с вариантами: `add`, `sub`, `mul`, `div` (по умолчанию `add`)
3. Принимает флаг **`--verbose` / `-v`**

### Продвинутые требования (со звёздочкой ★)
4. ★ Добавь флаг **`--precision` / `-p`** — количество знаков после запятой (по умолчанию 2)
5. ★ Добавь проверку: если операция `div` и `b == 0` — выдай понятную ошибку
6. ★ Добавь флаг **`--round`** — округлить результат до целого
7. ★★ Добавь субкоманду `history` — пусть хранит историю в памяти (пока просто выводит "История пока не реализована")

### Примеры запуска

```bash
# Базовый
cargo run -- 10 5
# → Результат: 15.00

# С операцией
cargo run -- 10 5 -o mul
# → Результат: 50.00

# Verbose + precision
cargo run -- 22 7 -o div -v -p 5
# → [INFO] Выполняется: 22 / 7
# → Результат: 3.14286

# Round
cargo run -- 22 7 -o div --round
# → Результат: 3

# Ошибка
cargo run -- 10 0 -o div
# → [ОШИБКА] Деление на ноль!

# Субкоманда
cargo run history
# → История пока не реализована
```

```rust
    command: Option<Commands>,

    #[arg(short, long, default_value = "add")]

}
```
